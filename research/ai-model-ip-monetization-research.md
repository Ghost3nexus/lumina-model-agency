## 1. サマリー

- **AIモデルのIP化は可能。ただし著作権ではなく商標権が主要な保護手段。** 日本の特許庁は2025年6月、AI生成のロゴ・マーク・キャラクターの商標登録を正式に認めた

## 2. 詳細

### A. 著作権の現状（日本法・米国法）

#### 日本法
| 項目 | 内容 |
|------|------|
| **原則** | AI「だけ」で生成した画像には著作権は発生しない（著作権法第2条：思想又は感情を創作的に表現したもの） |
| **例外** | 人間が創作的に関与した場合（プロンプト設計、加筆修正、選択・配置等）は著作物として認められうる |
| **文化庁見解** | 「AIを道具として利用して創作した場合」は従来の著作物と同様に扱う（2024年ガイドライン） |
| **確定判決** | AI生成物に関する著作権の確定判決はまだ出ていない（2026年3月時点） |

#### 米国法
| 項目 | 内容 |
|------|------|
| **原則** | 米国著作権局は「人間の著者性（human authorship）」を要求。純粋にAIが生成した作品は著作権保護の対象外 |
| **最高裁** | 2026年3月2日、「AIだけで著作物を創作できるか」の案件で上告棄却。人間の著者性要件を維持 |
| **例外** | AI支援で作成し人間が実質的に創作的表現を支配した場合は登録可能。著作権局はAI素材を含む作品の登録を数百件受理済み |

#### LUMINAへの適用
- CEOがファッションディレクターとしてプロンプト設計・画像選定・品質判断・加筆修正を行っている → **「人間が創作的に関与」に該当する余地あり**
- ただし著作権での保護は不安定。**商標権での保護が確実**

---

### B. 商標登録によるIP保護（最も確実な手段）

#### 日本の特許庁（2025年6月 方針確認）

**AI生成のキャラクター・ロゴ・マークは商標登録できる。**

| ポイント | 内容 |
|---------|------|
| **商標法の性質** | 商標は「創作物」ではなく「選択物」。著作権と違い、人間の創作性は要件ではない |
| **登録要件** | 従来と同じ。①識別力があること ②他人の登録商標と類似しないこと ③公序良俗に反しないこと |
| **AI関与の影響** | なし。AI生成であるかは商標登録の判断に影響しない |
| **保護範囲** | キャラクター名（文字商標）+ ビジュアル（図形商標）の両方を登録可能 |

#### 登録コスト（概算）

| 項目 | 費用 | 備考 |
|------|------|------|
| 出願料 | ¥12,000〜 | 1区分あたり（電子出願） |
| 登録料 | ¥32,900〜 | 10年分（1区分） |
| 弁理士費用 | ¥30,000-80,000 | 1出願あたり（調査込み） |
| **1体あたり合計** | **¥30,000-50,000** | 自力出願の場合。弁理士依頼で¥80,000-120,000 |
| **14体全体** | **¥420,000-700,000** | 自力出願の場合 |

#### 登録すべき区分
- **第35類**: 広告、モデルの提供（タレント派遣）
- **第41類**: エンターテインメント、ファッションショー
- **第25類**: 被服（将来のコラボアパレル展開時）
- **第9類**: デジタルコンテンツ（NFT等の将来展開時）

## 3. TomorrowProofへの示唆

### LUMINAの圧倒的優位性

1. **14体のロスターを持つAIモデルエージェンシーは日本に存在しない** — imma（Aww）は1-3体、AI model株式会社は企業広告特化。「事務所としてロスターを持つ」形態はブルーオーシャン
2. **技術コストが競合の1/100** — Aww（CGI制作）は1体の制作に数百万円。LUMINAはGemini APIで1体$1-5で生成
3. **スケーラビリティ** — 新モデル追加がほぼゼロコスト。100体でも可能
4. **Studio連携による二重収益** — モデルIP収益（ライセンス）+ Studio SaaS収益（月額）

### リスク

1. **著作権での保護は不安定** — 判例が確立していない。商標権が確実
2. **SNSフォロワー獲得に時間がかかる** — immaでも数年。短期の収益化は困難
3. **「AIモデル」へのバッシングリスク** — Forever 21, Mango等の事例。品質が低いとブランド毀損
