# Product Requirements Document (PRD)

## プロダクト名 (仮)
**Lumina High-End EC AI Photo Generator**

## 1. プロダクト概要
スマートフォンの簡単な写真（物撮り・平置きなど）から、カルティエ(Cartier)やティファニー(Tiffany)のようなハイブランド水準の超高品質なEC商品画像を生成するプロンプトベース/AI画像生成アプリケーション。メイン画像のサムネイルやキャンペーン広告レベルで即座に使用できる画質と構図を目的とする。

## 2. ターゲットユーザー
- アパレル、ジュエリー、小物のEC事業者
- 高品質なビジュアルを少ないリソース（時間・予算）で制作したいブランドオーナーやマーケター
- 商品の魅力を最大限に引き出したいが、プロのカメラマンやスタジオ手配が難しいユーザー

## 3. 解決する課題
- **コストと時間の削減**: プロのモデル、カメラマン、スタジオ手配にかかる費用とリードタイムの削減。
- **品質の底上げ**: スマホで撮影した素人写真でも、ハイブランドに匹敵する高級感のあるビジュアルを作成。
- **一貫性の担保**: ブランドイメージに沿ったトーン＆マナー（照明、背景、構図）をAIで一貫して再現可能。

## 4. ユーザーフロー（コア体験）
1. **情報入力**:
   - 商品の画像（スマホ撮影の平置き、トルソー、着用画像など複数枚）をアップロード。
   - 商品情報（素材、サイズ、ブランド名、カテゴリなど）をテキストで入力。
2. **スタイル選択**:
   - 用途（ECサムネイル、キャンペーンバナー、SNS用など）を選択。
   - 希望するライティング・雰囲気（カルティエ風のドラマチックな照明、ティファニー風のクリーンな照明など）を選択またはテキストで指定。
3. **AI生成・解析**:
   - AIがアップロードされた商品画像から形状、素材感、ディテールを解析。
   - 指定されたスタイルと商品情報を掛け合わせ、高品質な着用画像または物撮り画像を生成。
4. **確認と編集**:
   - 生成された複数パターンの画像を確認。
   - 必要に応じて、微調整（ライティングの強さ、背景の変更など）を指示して再生成。
5. **出力・保存**:
   - 高解像度でのダウンロード、またはECサイトへの直接連携。

## 5. コア要件・機能

### 5.1 入力インターフェース
*   **画像アップロード機能**:
    *   複数画像のアップロード（正面、側面、背面、ディテールなど）。
    *   スマホで撮影された低～中画質画像の許容と補正前処理。
*   **テキスト/メタデータ入力フォーム**:
    *   素材 (Material): ゴールド、レザー、シルク、コットンなど（質感表現に必須）。
    *   サイズ (Size): 実寸大のスケール感やフィット感をAIに理解させるための情報。
    *   ブランド/アイテム名: プロンプトのコンテキスト強化用。

### 5.2 生成AIエンジン・プロンプト制御
*   **画像解析（Vision AI）**:
    *   アップロードされた画像から商品の正確な形状、テクスチャ、色を抽出・保持。
*   **高品質画像生成（Generation AI）**:
    *   最高クオリティのプロンプトエンジニアリングにより、「ハイブランドのEC品質」「プロの物撮り・ライティング」を再現。
    *   **スタイルプリセット**:
        *   `Luxury Dramatic` (カルティエ風の陰影と高級感)
        *   `Clean Elegance` (ティファニー風の白銀・透明感)
        *   `High-Fashion Editorial` (雑誌のキャンペーン風)
        *   `Premium EC Studio` (純白/単色背景の最高級物撮り)

### 5.3 UI/UX要件
*   **シンプルで高級感のあるデザイン**:
    *   ターゲット層に合わせた、余白を活かしたミニマルでラグジュアリーなUI（ダークモードまたは洗練されたライトモード）。
*   **プレビュー＆比較機能**:
    *   生成前（オリジナル）と生成後（AI）のギャラリー表示。
    *   生成途中や処理状況の視覚的フィードバック（スケルトンローディング、プログレスバー等）。

## 6. 技術スタック案（現行 Lumina をベースとする場合）
- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS
- **AI Engine (Vision/Generation)**: 
  - Gemini 2.0 Flash (画像解析による商品ディテールの抽出)
  - Gemini 3 Pro Image または同等の高品質画像生成AI（Stable Diffusion 3, Midjourney API等、商品の形状保持力が極めて高いモデルの選定が必要）
  - ※特に「商品の正確な再現（ControlNet的な形状保持）」が最重要課題となるため、単なるプロンプト生成以上のパイプライン（Image-to-Imageの精緻な制御）が必要。
- **Styling**: `Lumina Fashion Studio` で培った洗練されたCSS/Tailwind変数流用。

## 7. 「Lumina Fashion Studio」との違い
*   **対象**: 
    *   Lumina: 服（アパレル）の着用モデル生成が中心。
    *   New App: アパレルに加え、**ジュエリー、小物、バッグ等の「物撮り」** や、よりハイエンドなキャンペーン品質に特化。
*   **クオリティ要求**:
    *   Lumina: リアルな人間モデルによるバリエーション展開。
    *   New App: 「カルティエ」「ティファニー」レベルの、**ライティング・質感・高級感への極限のこだわり**。商品のテクスチャ（金属の反射、革のシボ感など）の完全再現。

## 8. 成功の定義・KPI
- 生成画像のECサイト本番採用率（サムネイル/キャンペーン画像への使用）。
- 従来のプロ撮影コストに対する削減率（ROI）。
- ユーザーあたりの生成回数およびリピート率。
