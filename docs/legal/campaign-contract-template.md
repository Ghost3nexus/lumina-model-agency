# LUMINA Campaign / Exclusive Tier — Written Contract Template(骨子)

> Campaign / Exclusive tier 用の書面契約テンプレート **目次レベル**。
> 本文(各条項の具体文言)は案件個別に弁護士と最終化する前提。CloudSign または freeeサインで電子締結。
>
> ドラフト v0.1 · 2026-04-20 · 弁護士レビュー後に v1.0

---

## 使用シーン

- **Campaign tier**: シーズンキャンペーン、広告ブリーフ、OOH/印刷/放送含む案件
- **Exclusive tier**: カテゴリ独占(例: 「女性用スニーカー 6ヶ月間 ブランドA 独占」)

クリックラップでは法的リスクが高すぎる案件のため、**書面契約 + 電子署名**(CloudSign 推奨、freeeサインでも可)で締結。

## 契約書 Full Title

```
LUMINA MODEL AGENCY モデル使用ライセンス契約
— Campaign / Exclusive Tier —

Contract No.: LMA-2026-XXX
契約日: YYYY年MM月DD日
```

## 条項目次(15条 + 別紙)

### 第1条(定義)
本契約で使用する用語の定義(甲/乙 = TomorrowProof / クライアント、「本モデル」「本キャンペーン」「使用範囲」「Territory」「独占性」「Character Bible」等)。

### 第2条(本契約の目的・ライセンス付与)
- 付与される使用ライセンスの性質(非独占 / カテゴリ独占)
- 使用目的の具体化(キャンペーン名、商品ラインナップ、期間)
- ライセンスが包含する使用範囲:
  - 媒体カテゴリ(Web / SNS / 印刷 / OOH / 放送 / OTT / その他)
  - 具体的なプラットフォーム名(Meta / Google / TikTok / LINE Ads / YouTube / 雑誌名 / etc.)
  - 地域(日本 / グローバル / 限定地域)
- 本契約に基づき**付与されない権利**の明示(サブライセンス、未指定地域への展開、未指定媒体への転用)

### 第3条(独占性 · Exclusive tier 限定)
- 独占カテゴリの定義(例: 「女性用スニーカー」「メンズ腕時計」)
- 独占期間中の当社の制約(同カテゴリ内 他社へ同モデルを貸し出さない)
- カテゴリ範囲の境界事例処理(隣接カテゴリでの他社使用可否)
- 独占違反時の補償金条項

### 第4条(Creative Direction · クリエイティブの決定権)
- 当社側のモデルキャラクター・Character Bible への変更拒否権
- クライアント側のキャンペーン方向性・スタイリング・起用シーンの決定権
- 両者協議事項(撮影コンセプトの逸脱判定、ブランドセーフティ懸念事項)

### 第5条(成果物仕様・納品物)
- 画像: 枚数 / 解像度 / フォーマット(PNG / JPEG / WebP)/ モール規格対応の有無
- 動画: 本数 / 尺 / 解像度 / フォーマット(MP4 H.264/265)
- 修正可能回数(例: 3回まで無償、以降は別途見積)
- 納期(日付ベース or 営業日ベース)
- 納品形式(Supabase storage / ZIP ダウンロード URL / Google Drive 共有)

### 第6条(料金・支払い)
- 基本料金(一括前払い or 分割)
- 追加料金発生条件(修正超過、納期短縮、追加ショット)
- 消費税・源泉徴収(ブランド側が源泉徴収義務者の場合)
- 支払い期日(納品後 30 日 or 請求書発行から 30 日等)
- 遅延損害金(年 14.6%)

### 第7条(知的財産権)
- LUMINA が保持する権利(モデル IP、Character Bible、生成パイプライン、原始出力物)
- クライアントが保持する権利(提供商品画像、ブランドロゴ、クリエイティブ指示書、編集済みコンポジット)
- ジョイント取扱い事項(Campaign 独自コンセプト等)
- LUMINA のケーススタディ利用権(クライアント同意の上で公開可)

### 第8条(実在人物非類似性の保証)
- LUMINA は本モデルが実在人物(存命・故人問わず)と識別可能に類似しないよう合理的な注意を払う
- クライアントは本モデルを特定の実在人物の代用として使用してはならない
- クライアントが実在人物類似を主張する第三者クレームを受けた場合、LUMINA が対応責任を負う範囲

### 第9条(秘密保持)
- 別途 NDA が締結されている場合はそれに従う
- NDA がない場合は本条を適用(相互秘密保持、存続期間 3 年)
- 未発売商品情報・キャンペーン計画・価格情報・ブリーフ資料を秘密情報として取り扱う

### 第10条(表明・保証)
- LUMINA: 本モデルの IP を保有していること、第三者権利を侵害しないこと
- クライアント: 提供素材が第三者権利を侵害しないこと、反社会的勢力に属さないこと
- 両当事者: 法令遵守、適切な権限者による契約締結

### 第11条(免責・損害賠償)
- LUMINA の責任範囲(契約金額の 2 倍を上限 等、業界標準)
- 逸失利益・間接損害の免責
- 故意・重過失の場合の例外
- 保険加入状況(将来: E&O保険)

### 第12条(契約期間・更新)
- 契約開始日・終了日
- Exclusive の場合の自動更新条項(6 ヶ月ごとの更新、90 日前通知)
- Campaign の場合の特例(キャンペーン終了時が契約終了)

### 第13条(解除)
- 期限の利益喪失事由(支払い遅延、破産申立等)
- 重大違反時の即時解除権(本モデルの実在人物化、無許諾サブライセンス等)
- 催告・治癒期間の設定
- 解除時の既納品の扱い(原契約範囲内の継続使用可)
- 未履行部分の料金返還有無

### 第14条(その他)
- 通知方法(メール + 書面 双方向)
- 権利譲渡禁止
- 不可抗力条項(天災、戦争、重大な法改正)
- 部分無効(1 条項の無効が全体に影響しない)
- 完全合意(他の合意との優先順位 — 本契約 > Click-wrap > 利用規約)

### 第15条(準拠法・紛争解決)
- 準拠法: **日本法**
- 第一審: **東京地方裁判所の専属的合意管轄**
- 国際 Exclusive 契約の場合: Japan Commercial Arbitration Association を選択肢として
- 紛争発生時の事前協議期間(30 日の善意交渉)

## 別紙

### 別紙 A — 契約モデルリスト
- モデル名 / Division / 主要 Character Bible 要素(外見・比率・声質・スタイリング)
- 参照 URL(社内 Character Bible 公開版)

### 別紙 B — 使用範囲詳細表
- 媒体別 × プラットフォーム別 マトリクス
- ✅ 許諾、❌ 禁止、△ 要事前協議 をセル単位で明記

### 別紙 C — 成果物仕様書
- 画像スペック表(枚数・解像度・フォーマット・納期)
- 動画スペック表(本数・尺・フォーマット・納期)
- 修正可能回数・対応範囲

### 別紙 D — 料金明細
- 基本料金(内訳: モデル IP 使用料 + 制作費)
- 追加料金の単価表
- 支払いスケジュール

### 別紙 E — 禁止表示リスト
- AI Ethics Code `§4` を契約別紙として添付
- 当該契約で特に注意すべきブランドセーフティ要件(業界別追加禁止事項)
