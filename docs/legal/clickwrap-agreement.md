# LUMINA Click-wrap Agreement — Standard / Extended tier 用

> 電子消費者契約法・電子署名法準拠の電子合意フロー骨子(B2B想定のため電子消費者契約法の錯誤特例は限定的)。
> **CEO決定(2026-04-20)**: 全 tier B2B商談ベースのため、本クリックラップは**契約締結段階**で使用(購入時ではなく、商談・提案後に契約成立の電子同意として取得)。
> ドラフト v0.1 · 2026-04-20 · 弁護士レビュー後に v1.0

---

## 1. 使用シーン

Sales商談→提案・見積書受諾→**オンライン契約ページでクリックラップ同意**→ freee で請求書発行 → 月次課金開始。

```
商談 30分
  ↓
提案書(PDF) + 見積書(freee)送付
  ↓
クライアント専用 URL で本同意文表示
  ├ チェックボックス: [ ] 上記規約および個別契約内容に同意します
  ├ テキスト入力: 会社名・担当者名・メールアドレス
  └ [ 同意して契約を締結する ] ボタン
  ↓
サーバー側 log 保存(IP / UA / timestamp / 同意版数)
  ↓
確認メール送信(Resend)
  ↓
Sales に Slack 通知
  ↓
freee 請求書発行(初回)
```

## 2. 日本法準拠ポイント

| 法律 | 対応箇所 |
|---|---|
| **電子署名法 第2条** | 「電磁的記録」に本人の識別可能情報(メール/ID)付きで同意ログ保存 |
| **電子消費者契約法 第3条** | 誤操作防止のため、同意前に全条項表示 + 確認画面 2段階(B2B中心だが未成年/個人事業主が顧客となる可能性を考慮) |
| **電子帳簿保存法** | 同意ログを7年保存、タイムスタンプ付与 |
| **特定商取引法 第11条** | B2Bは基本適用外だが、個人事業主への販売時は明示義務あり→ 常に特商法表記を表示 |

**第1条(本契約の対象)**
1. 当社は、お客様に対し、LUMINA MODEL AGENCY の AI モデル(以下「本モデル」)のうち、別紙「契約モデルリスト」に記載のモデルを、本契約で定める範囲で使用する非独占的・譲渡不可のライセンス(以下「本ライセンス」)を付与します。
2. 本ライセンスは、本契約の期間中、以下の使用範囲に限り有効です。
   - **Standard**: お客様自身が運営する EC 商品ページ(楽天、Yahoo、Amazon、Shopify、自社直販サイト)における商品画像としての使用のみ。
   - **Extended**: Standard の範囲 + お客様自身が運営する SNS アカウント、ブランドブログ、編集コンテンツ、プロモーショナルメールにおける使用。
3. 使用可能な画像・動画の**月間上限**は以下のとおり。
   - Standard: 画像 月 100 枚 / 動画 なし
   - Extended: 画像 月 300 枚 / 動画 月 10 本(各 15秒以内)

**第2条(許諾されない行為)**
お客様は、本ライセンスの範囲外で次の行為を行ってはなりません。
1. 有料広告(Meta / Google / TikTok / LINE / YouTube / OOH / 印刷 / 放送 / OTT)における本モデルの使用。必要な場合は別途 **Campaign tier** 契約を締結する必要があります。
2. 第三者へのサブライセンス、再譲渡、リセラー経由の配布。
3. 本モデルを実在人物として誤認させる表示、または「人間モデルが撮影した」と虚偽表示すること。
4. 本モデル自体を商標または独自IP として登録する試み。
5. 本モデルを政治活動、宗教的勧誘、未成年の性的描写、違法行為の助長、ヘイトスピーチ、各広告プラットフォームのブランドセーフティ違反目的で使用すること。

**第3条(知的財産権)**
1. 本モデルのアイデンティティ、名称、肖像、声、キャラクター属性に関する一切の著作権、キャラクター権、その他の知的財産権は、**当社が全世界・永続的に排他的に保有**します。
2. お客様が本契約期間中に作成・発注したコンポジット成果物(編集済み画像・動画)は、本モデル素材の使用範囲内でお客様が使用できますが、本モデル素材そのものの権利はお客様に移転しません。
3. お客様が当社に提供する商品画像・ロゴ・ブリーフ資料等の知的財産権はお客様に帰属し、当社は本契約範囲に限りこれを使用します。

**第4条(契約期間・更新)**
1. 本契約の期間は、契約締結日から **1 か月** とします。
2. 期間満了の **30 日前** までにいずれかの当事者から書面による不更新の申し出がない限り、同一条件で自動的に 1 か月ごとに更新されます。
3. 月途中での解約による日割り返金は行いません。

**第5条(料金・支払い)**
1. 料金は以下のとおりです(消費税別)。
   - Standard: **月額 5,000 円 / モデル 1 体あたり**
   - Extended: **月額 15,000 円 / モデル 1 体あたり**
2. 当社は毎月所定の締め日に請求書を発行し、お客様は請求書記載の期日までに銀行振込またはクレジットカード決済により支払うものとします。
3. 支払い遅延が生じた場合、年 14.6% の遅延損害金を請求できます。

**第6条(納品物の範囲)**
本ライセンスに含まれる画像・動画の生成は、お客様から別途 **コンテンツ制作依頼(Content Production)** を受けた場合に行い、依頼は案件ごとに個別に受諾または見積を提示します。本ライセンスそのものは、生成作業の前払いではなく、モデル使用権の月額料金です。

**第7条(秘密保持)**
1. 両当事者は、本契約履行の過程で知り得た相手方の営業上・技術上の秘密情報を、第三者に開示または漏洩してはなりません。
2. 本条項は、契約終了後も **3 年間** 存続します。

**第8条(反社会的勢力の排除)**
両当事者は、暴力団・暴力団員・暴力団準構成員・その他の反社会的勢力に属さないこと、資金提供・便宜供与を行わないことを表明・保証します。違反時、相手方は催告なしに本契約を解除できます。

**第9条(契約解除)**
次のいずれかに該当する場合、相手方は催告の上(第2条 5 号・第8条違反は催告なし)本契約を解除できます。
- 重大な契約違反があり、30 日以内に治癒されないとき
- 支払い遅延が 30 日を超えたとき
- 破産・民事再生手続等の申立てがあったとき

**第10条(解除時の効果)**
1. 解除時点で既に適法に公表・納品された画像・動画は、お客様が引き続き原契約範囲内で使用できます。
2. 解除後の新規生成、未公表素材の新規公表、または原契約範囲外への使用拡大は**直ちに停止**するものとします。

**第11条(モデル引退時の措置)**
1. 当社は、倫理的理由、創作者の離脱、Character Bible の抜本改訂等により、本モデルを引退(Retirement)させる場合があります。
2. 引退の場合、**60 日前**までにお客様に通知します。
3. 引退後、お客様は新規生成依頼を行えませんが、既納品物は原契約範囲で継続使用可能です。

**第12条(免責)**
当社は、本モデル使用により生じた第三者との紛争・損害について、**当社に故意または重大な過失がある場合を除き**、一切の責任を負いません。お客様は自己の判断と責任で本モデルを使用するものとします。

**第13条(準拠法・管轄)**
本契約は**日本法**に準拠し、本契約に関する一切の紛争は**東京地方裁判所を第一審の専属的合意管轄裁判所**とします。

**第14条(本契約と関連文書との関係)**
本契約は、当社の以下の文書と一体として解釈されるものとします。

## 4. 技術実装メモ(Dev向け)

- 同意ログ保存先: Supabase `legal_consents` テーブル
  - `client_id`(uuid) / `agreement_version`(text) / `tier`(enum) / `models`(text[]) / `ip`(inet) / `user_agent`(text) / `consented_at`(timestamptz) / `contract_pdf_url`(text)
- 同意版数(`agreement_version`)を変更した場合、既存クライアントには再同意を求める or 旧版を維持(契約期間中は旧版保護)
- 同意ページは `/contracts/[token]`(トークンは Sales が商談後に発行)
- 契約書 PDF はクリックラップ後、自動で CloudSign or PDF生成ライブラリで発行し S3/Supabase storage に保存
