## 3. Creative Direction — KOZUKI TAKAHIRO

### なぜこのキャラクターを創造したか

LUMINA MODEL AGENCY の ladies-intl / ladies-asia 6体はすべて 174-181cm の「撮影規格身長」。これは EC とエディトリアルには最適だが、**SNS インフルエンサー案件には身長が高すぎて機能しない**。Reels / TikTok / Instagram の世界では、165cm 前後の「親しみやすい身長」「友達感覚の距離」が圧倒的にエンゲージメントが強い。RINKA はこの SNS 軸専用に内製したキャラクター。**165cm という身長はハウスで最低**(KAI 178cm を除くと RINKA だけが明示的に低身長)で、これは意図的な設計 — モデルではなくインフルエンサーとして機能させるため。

ピンクアッシュのヘアと下北沢の空気感を運ぶ vibe は、ZOZOTOWN、Shein、Uniqlo、GU、原宿ストリートブランドの SNS 運用案件で刺さる設計。GRWM(Get Ready With Me)、OOTD、Vlog、HAUL 動画の全てが自然に成立する。

### 差別化ポイント(他ロスターとの関係)

- **vs MIKU**: MIKU は日本の静謐・前衛、RINKA は日本のストリート・ポップ。同じ日本女性でも全く別レイヤー
- **vs KAI**: KAI と同じインフルエンサー軸。男女ペアとして成立、しかし KAI は Byron Bay 感、RINKA は下北沢感と明確に分ける
- **vs ladies-intl / ladies-asia**: 撮影モデルではなく SNS クリエイター。tier 別ライセンスの建付けも別想定
- RINKA は **LUMINA で唯一、"モデル規格を外れた身長"** をあえて持つ ladies キャラクター

### スタイリング・振る舞いの原則

- 表情は自然、友達感覚。笑顔を多用、セルフィー的
- 立ち姿はラフ、ピース、カジュアルな小物遊び
- ヘアはピンクアッシュ、ショルダー長、ストレート〜ゆるウェーブ
- アクセサリーは遊び心(ヘアクリップ、パールバレッタ、シルバー、古着的ヴィンテージ小物)

### 顔・体の特徴的な設計判断

- **Pink-ash hair**: ブリーチからの被せピンク、原宿・下北カルチャーの sig。毎月の色の揺らぎも演出できる
- **165cm**: SNS インフルエンサー身長。ハイモデル規格から意図的に外す
- **W58**: ladies の中で最細。スリムな現代日本女性の平均的スリム像

### 採用・起用シーン

- ZOZOTOWN、Shein、Uniqlo、GU、アパレル EC の SNS 運用素材
- 化粧品・コスメの SNS 訴求(プチプラ〜中価格帯)
- 原宿・下北沢ストリートブランドのインフルエンサー起用
- Reels / TikTok / YouTube Shorts の GRWM / OOTD / HAUL 動画
- カフェ・スイーツ・ライフスタイル広告
- V-POP アイドル文脈に寄せた案件
