## カテゴリ別 Top 3 アプローチ戦略

### 1. ファッション D2C ストリート・セレクト(#1-5, #18-20)
- **訴求軸**: Character Bible(人物起用の延長線として理解しやすい)+ 継続起用可能性
- **見せる画像**: LUCAS MORI(BEDWIN muse) / RINKA(harajuku creator)
- **温度感**: 業界リファラル重視。PR TIMES・展示会・IG 相互フォローから温めて入る
- **初回オファー**: 30分 Google Meet で Lookbook SS26 ウォークスルー + 1モデル無償トライアル提案

### 2. アウトドア・ライフスタイル(#9-11)
- **訴求軸**: 季節ごとのシーズンキャンペーン撮影コスト(ロケハン + 天候リスク)を AI で完全制御
- **見せる画像**: ELENA(Nordic / Scandinavian editorial)/ LUCAS MORI(アメトラ × アウトドア)
- **温度感**: ECの回転率を数値で語る。「月20SKU × 年間240 SKU で撮影コスト¥36M → ¥18万」の削減事例(pricing-rationale Axis 4)
- **初回オファー**: 既存 SKU 5点を使った無償サンプル生成(24時間納品)

### 3. ラグジュアリー / コスメ / 小物(#6-8, #12-15)
- **訴求軸**: Exclusive tier(カテゴリ独占)+ AI Ethics Code(ブランドセーフティ)。「AIで安く」ではなく「ブランドIPを守る新しいエージェンシー」
- **見せる画像**: ELENA(editorial luxury)/ 全モデル beauty.png 統一トーン
- **温度感**: プライス訴求禁止。Character Bible × IP tier × Ethics Code の3柱で品格を担保
- **初回オファー**: 60分商談 + 既存ブランドガイドラインに合わせた Character Bible カスタマイズ提案

## 注意事項

- **実在企業名を使用しているが、売上は業界推計ベース**(IR非開示社が大半)。商談時は必ず直近 PR TIMES / IR を再確認。
- **Contact 宛先は公式サイトで記載されている一般問合せ窓口**。実際の PR / マーケ決裁者名は商談ドア開き後に個別特定。
- **A優先度でも BEDWIN 以外は初回「ご挨拶」温度**。いきなり契約提案せず、ブランド理解を示した上で商談アポを取る。
- **大手(sacai / PORTER / ASICS 等 C優先度)は Launch 初月に動かない**。実績(BEDWIN 26SS + 中堅D2C 3社)を積んでから。
