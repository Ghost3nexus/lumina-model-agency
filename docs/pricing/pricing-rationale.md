# LUMINA MODEL AGENCY — Pricing Rationale

> 4-tier License pricing (Standard ¥5,000 / Extended ¥15,000 / Campaign Inquire / Exclusive from ¥200,000) の価格根拠を **4軸**(コスト積み上げ / 市場比較 / LTV / 削減効果) で証明する。
>
> 関連: `docs/design/ip-license-tiers.md` / `docs/design/MASTER-claude-design-prompt.md` §6
> ドラフト: 2026-04-20 / 承認: 未 (CEO確認後 v1.0)
> 根拠データ最終確認: 2026-04-20

---

## 0. Executive Summary

LUMINA の 4-tier 価格は以下 4軸で説明・防衛可能にする。

1. **コスト積み上げ**: Standard 実コスト 〜¥1,000 → マージン80% (SaaS健全レンジ)
2. **市場比較**: Botika $22/月の1.5倍(premium positioning)、ai-admakers ¥833/月の6倍(品質・IPの差)
3. **LTV**: Standard 2年 ¥120k / Extended 2年 ¥360k / Exclusive 1年 ¥2.4M
4. **削減効果**: 従来EC撮影 年間¥18-36M → Lumina 年 ¥60k〜1.8M、**80-99% 削減**

顧客への価格説明は **4軸のどれでも答えられる**状態にする。競合より高い質問には (1)(2)、高すぎる質問には (4)、ROI質問には (3)(4) で答える。

## 1. 4-tier 構造の設計ロジック

| Tier | 月額 | 想定顧客 | 選定根拠 |
|---|---|---|---|
| **Standard** | ¥5,000 / model | 小規模D2C(月商〜5M) | 従来撮影¥85-300k/回(lumina-studio-lp 明記)の **1/10 以下** で始められるエントリー。friction最小 |
| **Extended** | ¥15,000 / model | 中堅ブランド(月商 5-50M) | 画像300枚+動画10本 = ささげ+SNS月額パッケージ相当(従来¥150-400k/月)の **1/15-1/20** |
| **Campaign** | Inquire | シーズン施策がある大手(月商 50M+) | プロジェクト単位。**特商法上、価格明示を避けるため非公開**(法的余地確保) |
| **Exclusive** | From ¥200,000 / model / mo | ブランドコラボ(月商 100M+) | カテゴリ独占の対価。人間モデル専属契約(¥500k-3M/月)の **1/3 以下** で独占権を販売 |

### 価格格差のロジック
- Standard → Extended: **3倍**(SNS+Web許諾で使用範囲 3倍相当、3倍の金額が自然)
- Extended → Exclusive: **13倍**(独占性の premium、競合排除コスト内包)
- Campaign は line item のため構造外(見積ベース)

## 2. 証明責任 Axis 1 — コスト積み上げ

### Standard tier (¥5,000/月) の単位コスト

| 項目 | 月あたり | 根拠 |
|---|---|---|
| 画像生成 (Gemini 3 Pro Image Preview) | ~¥150 | $0.02-0.05/枚 × 100枚上限 = $2-5 ≒ ¥300-750 想定、実際は生成数少なめで抑制 |
| 動画生成 (使用時のみ) | ¥0 (Standardは動画なし) | — |
| 配信・ストレージ (Vercel / Supabase) | ~¥50 | 1モデルあたりの CDN / DB 按分 |
| 人間QAレビュー (Tier 2) | ~¥500 | 月 5-10枚の目視確認、時給¥3,000 × 10分 按分 |
| 共通インフラ按分 (auth / 決済 / monitoring) | ~¥300 | SaaSコアコスト / 顧客数 |
| **単位コスト合計** | **~¥1,000** | |
| **粗利** | **¥4,000 (80%)** | SaaS健全レンジ(70-85%) 内 |

### Extended tier (¥15,000/月)
- 画像3倍 + 動画10本(月)
- 動画 Kling via Replicate: $0.35-0.95/本 × 10 = $3.5-9.5 ≒ ¥500-1,400
- QAレビュー拡大: ¥1,500 (枚数 3倍)
- 単位コスト 〜¥3,500 → 粗利 ¥11,500 (77%)

### Exclusive tier (¥200,000+/月)
- Unlimited generation: コスト変動大、月平均 ¥15,000-30,000 と想定
- 独占契約管理 + Legal 手間: 工数 ¥30,000/月
- 単位コスト 〜¥45,000 → 粗利 ¥155,000 (78%)

### Campaign
- 案件ベースで見積。工数 + 画像・動画コスト + マージン 50%+ で設計

### Insight
**全 tier で粗利 70-80% を維持**。これは Botika($22/月、推定マージン 60-70%)より高効率。
Character Bible 再利用、ロスター固定化、自動化QAにより**生産コストが低く抑えられる**のが強み。

## 4. 証明責任 Axis 3 — LTV(顧客生涯価値)

### 前提
- 解約率(月次): Standard 5% / Extended 3% / Exclusive 1% (予想、初年度は + 保守的に 1.5倍)
- 平均継続月数: 1/解約率 = Standard 20ヶ月 / Extended 33ヶ月 / Exclusive 100ヶ月

### Tier別 LTV

| Tier | 月額 | 平均継続 | 単純LTV | 複利加味 (参考) |
|---|---|---|---|---|
| Standard | ¥5,000 | 20ヶ月 | **¥100,000** | 〜¥90,000 (NPV 10%) |
| Extended | ¥15,000 | 33ヶ月 | **¥495,000** | 〜¥420,000 |
| Campaign (1案件) | ¥300k-2M | — | プロジェクト毎 | — |
| Exclusive | ¥200,000 | 100ヶ月 (但し現実的には12-24ヶ月) | **¥2.4M-4.8M** | 〜¥1.8M-3.6M |

### CAC(顧客獲得コスト)上限

LTV/CAC 比率 3:1 を健全基準とすると:

| Tier | 許容CAC |
|---|---|
| Standard | 〜¥33,000 |
| Extended | 〜¥165,000 |
| Exclusive | 〜¥800,000-1,600,000 |

→ 広告費・商談リソースは tier に応じて配分可能。Exclusive は Sales 人件費掛けても十分ペイする。

**前提**: 月10SKU × 年120SKU

| 項目 | 従来 | LUMINA Standard |

| 削減率 | — | **99.6%** |

### Campaign 単発比較

**前提**: シーズンキャンペーン 1本(3モデル × 10カット × 動画2本)

## 6. 前提・感度分析

### 価格感度(下振れリスク)
- Standard を¥3,980に下げた場合: 年LTV -20%、広告費上限縮小 → 獲得競争力増も収益性低下
- Extended を¥12,800に下げた場合: Standard→Extended アップグレード加速の可能性

### 価格感度(上振れリスク)
- Standard を¥7,980に上げた場合: ai-admakers 比 10倍 → 価格起点で離脱懸念。Character Bible 訴求強化が必須

### コスト上昇リスク
- Gemini API 値上げ 2倍 → Standard コスト¥300→¥600、マージン 88% → 76%(許容)
- 為替リスク(USD 10%円安): Gemini / Replicate コスト増 → Standard マージン -2%(軽微)

### 感度表(Standard)

| 月額 | 粗利 | 年LTV(継続24ヶ月想定) |
|---|---|---|
| ¥3,980 | 75% | ¥71,640 |
| **¥5,000 (現)** | **80%** | **¥100,000** |
| ¥6,980 | 86% | ¥131,640 |

→ ¥5,000 は **LTV/コスト/獲得競争力** のスイートスポット。

## 8. FAQ(営業用)

**Q: なぜ ai-admakers.jp(¥833/月)より 6倍高いの?**
A: LUMINA は各モデルに **Character Bible** を持ち、ブランドで継続起用できる「キャラクターIP」です。AI 生成画像を売る ai-admakers と、**キャラクターを育てて貸し出す**LUMINA は本質的に異なるサービスです。Character Bible による品質一貫性、AI Ethics Code による法的安全性、IP ライセンス明文化が価格に含まれます。

**Q: なぜ Botika($22/月)より高いの?**
A: Botika は月制作枚数と撮影時間を競うツールです。LUMINA は**ブランドの顔となる専属モデル**を貸し出すエージェンシーです。日本市場ネイティブ(JP/EN 両対応)、モール規格(楽天/Yahoo/Amazon)対応、Character Bible によるブランド継続性が Botika にはありません。

**Q: 撮影代替で80%-99% 削減って本当?**
A: 前提条件により変動しますが、月10-20SKU を回すブランドなら **確実に80%以上** の削減が出ます。中堅以上なら 99% 削減事例もあります。LUMINA 内部デモで 20SKU 比較を実施済み (BEDWIN + 20SKU EC 事例、`docs/design/ip-license-tiers.md` 参照)。

**Q: Campaign の価格はなぜ非公開?**
A: プロジェクトの規模・期間・使用範囲で大きく変動するため、個別見積としています。目安として **¥500,000 - ¥3,000,000 / 案件**、シーズンキャンペーンは後者帯が標準です。

**Q: 契約期間は?**
A: Standard / Extended は月次更新でロックインなし、30日前通知で解約可能。Exclusive は最低6ヶ月、以降6ヶ月ごとの自動更新で 90日前通知で解約可能です。

## 9. 価格改定のトリガー条件

### 下げる場合
- 解約率が月 8%超え(需要に対して価格が高すぎ)
- 競合が LUMINA 価格の 70% 以下で同等品質 (Character Bible + IP 保有) を提供開始

### 上げる場合
- 解約率が月 2%以下(需要過多で値上げ余地)
- 本番運用コストが想定の 1.5倍超(マージン防衛)
- ブランド格が確立(メディア露出、実績、大手導入)

## 10. 運用ルール

- **Launch時(2026-05-10)**: 本価格表で固定。**最低90日間は値動きさせない**(顧客信頼)
- **90日後レビュー**: 実データで仮説検証、必要に応じて Phase 2 price 再設計
- **価格変更時の既存顧客**: 契約期間中は旧価格維持、次回更新時より新価格適用
- **競合価格調査**: 月次で ai-admakers / Botika / 新規AIエージェンシーを監視

### ⚠️ 販売フロー(CEO 2026-04-20 確定、シンプル化)

**全 tier で B2B 商談ベース販売 + freee 請求書 + 銀行振込**。**Stripe / CloudSign / freeeサイン は Launch phase 1 で不採用**。

理由: (1) クライアント毎にモデル指名・使用媒体・成果物仕様が異なる → 即購入トラブル回避、(2) 電子契約ツール費は Launch 初期には過剰、(3) B2B 法人間商慣習として freee 請求書 + 銀行振込は十分。

| tier | 販売フロー | 契約方式 |
|---|---|---|
| Standard / Extended | `/for-brands` inquiry form → Sales 商談(30分) → 提案書(PDF) → メール合意または注文書返送 → **freee 請求書発行** → **銀行振込** → サービス開始 → 月次 freee 請求 | メール同意 + 内部契約テンプレート |
| Campaign | inquiry → Sales 商談(60分) → brief 確認 → 提案・見積書 → **案件発生時に弁護士アサイン**し書面契約 → freee 請求(案件単位、前金 or 納品後) | **書面契約**(弁護士レビュー済) |
| Exclusive | inquiry → 複数回商談 → NDA 締結 → **弁護士書面契約** → freee 請求(月次) | **書面契約**(弁護士フル関与) |

### 特記
- **Campaign / Exclusive は Launch phase 1 では "Inquire" 受付のみ**。実際の契約締結は BEDWIN 等実案件発生時に弁護士投入して対応
- 弁護士アサインは**需要見込時 or BEDWIN 始動シグナル検知時**に即発動(事前に候補 3 社ピックアップは推奨)
- `/studio` self-serve (Free/Starter/Pro/Business) は別動線で **現状の Stripe Payment Links 維持**。個人向けロングテール、For Brands 商談と切り離す
