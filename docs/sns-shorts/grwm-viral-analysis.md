## 構造テンプレート（共通パターン抽出）

### フック（0-3秒）

- **最頻出パターン**: Before状態（すっぴん/パジャマ/散らかった部屋）からスタート。視聴者に「この人がどう変わるか」の期待を持たせる
- **次点**: ASMR的な音（コスメの蓋を開ける音、コーヒーを注ぐ音、服のタグのカサカサ音）で五感を刺激
- **第3位**: テキストフック（「今日のコーデ迷う」「$200 haul」「charity shop find」等、情報提示で好奇心を刺激）
- **RINKA向け推奨**: **パジャマ姿のダル顔 → ピンクアッシュ髪をかき上げる → 「今日はsacai」テキスト**。理由: ストリート系はBefore/Afterのギャップが最大の武器。ピンクアッシュ髪という視覚的フックが既にある。ブランド名テキストでファッション感度の高い層を引き留める

### カット割り

- **平均カット長**: 2-3秒/カット
- **平均総カット数**: 10-14個（30秒尺の場合）
- **テンポ変化パターン**: 序盤ゆっくり（3-4秒/カット） → 中盤加速（1.5-2秒/カット） → ビートドロップで瞬間変身 → 完成ルックはやや長め（3-5秒）
- **最短成功パターン**: 15秒 / 4-6カット（outfit transition特化型）
- **完走率データ**: 15秒以下 = 68%完走率、60秒超 = 48%完走率

### カメラアングル比率

- 鏡越し全身: 45%（最も多い。統一感と「リアルな自撮り」感を両立）
- 三脚正面: 20%（メイクパート中心）
- 手持ち/セルフィー: 10%（トーク系 or 外出パート）
- クローズアップ（手元/目元/足元）: 20%（ディテール見せ。エンゲージメント向上に直結）
- 俯瞰（フラットレイ）: 5%（コーデ全体を見せる導入に効果的）

### テキスト

- **フォント**: サンセリフ太字が主流。日本向けはゴシック体（丸ゴシック or 角ゴシック）。ストリート系はグラフィック的な太字
- **配置**: 中央上部（Safe Zone: 上端200px-600px）。顔やキーアイテムに被せない
- **出現頻度**: 3-5箇所/動画（フック + アイテム名2-3個 + 締め）
- **内容パターン**:
  - 情報系: ブランド名、価格、入手先（EC購買層に刺さる）
  - ツッコミ系: 「これで¥500」「3時間悩んだ」（共感と笑い）
  - CTA系: 「保存して」「どっちがいい?」（エンゲージメント誘導）
- **技術仕様**: 最低36pt相当 / 白文字+2px黒ストローク / 表示時間1.5-3秒 / フェードイン0.3秒

### トランジション

- **最頻出**: ジャンプカット（全体の70%以上）。シンプルだが音楽のビートに合わせることで印象的になる
- **効果的だった例**:
  - **ビートドロップ outfit change**: 音楽のビートドロップに合わせ、同じ位置・同じポーズで服だけ変わる。align機能で最後のフレームを表示し位置合わせ
  - **手かざしトランジション**: 手でカメラを覆う → 覆いを外すと変身完了
  - **スピントランジション**: 回転中にカットし、回転後に別コーデ
  - **服引っ張りトランジション**: 上着の裾を引っ張る → 新しい服に変わる
- **避けるべき**: 過度なディゾルブやグリッチエフェクト（2024年以前のトレンド。2026年はクリーンカット主流）

### 尺配分（30秒の場合の推奨）

| セクション | 時間 | 比率 | 備考 |
|-----------|------|------|------|
| フック | 0-3秒 | 10% | Before or テキスト or ASMR音 |
| メイク/スキンケア | 3-10秒 | 23% | 入れない場合はコーデ選びに充てる |
| コーデ/着替え | 10-22秒 | 40% | メインコンテンツ。カット頻度最高 |
| 完成ルック+締め | 22-30秒 | 27% | 外出ショット or 360度回転 or ポージング |

### BGM

- **ジャンル傾向**:
  - ストリート系: Hip-Hop / Trap / J-HipHop（130-150BPM）
  - エステティック系: Lo-fi / アンビエント / シティポップ（80-110BPM）
  - トランジション特化: ビートドロップのある楽曲（テンポ不問、ドロップの明確さが重要）
- **2026年4月トレンド楽曲**:
  - Charli XCX "360" orchestral cover（Bridgerton風。ドラマティック変身に最適）
  - Ella Langley x Fleetwood Mac mashup（outfit transition + aspirational lifestyle）
  - Tyla（ファッションGRWM + outfit change向き）
  - "Lush Life"（ファッショントランジション + グローアップ）
  - Bruno Mars "Cha Cha Cha"（軽快な日常GRWM向き）
  - "Carefree Days"（朝ルーティン系GRWM向き）
- **テンポ帯**: ファッションGRWMは110-140BPMが最適。遅すぎると退屈、速すぎると落ち着かない
- **重要**: トレンド音源は発見後24-48時間以内に使用。TikTokで先行3-7日のものがIG Reelsでの穴場
