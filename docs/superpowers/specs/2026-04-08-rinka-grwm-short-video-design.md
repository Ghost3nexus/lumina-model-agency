# RINKA GRWM Short Video — Design Spec

> LUMINA MODEL AGENCY AIインフルエンサー SNSショート動画 第1弾
> 対象モデル: RINKA（influencer-girl-01）
> 投稿先: Instagram Reels（9:16）
> 承認: 2026-04-08 CEO承認済み

---

## 1. 目的

LUMINA MODEL AGENCYのAIモデルをSNSインフルエンサーとして活動させる。
第1弾としてRINKAのGRWM（Get Ready With Me）動画を1本制作し、パイプラインを検証する。

### 品質基準

**「AIとわからない」レベル**。人間インフルエンサーのGRWMと並べて違和感がないこと。
社長が「これは投稿できる」と判断するまで未完了。

---

## 2. 全体フロー

```
① 競合バイラルGRWM分析（構造抽出）
② RINKA用GRWMコンセプト設計（台本・カット割り）
③ 静止画生成（Gemini → 8-12カット）
④ 動画変換（各カットを Kling/Runway で2-4秒動画化）
⑤ 編集・仕上げ（CapCut でつなぎ + テキスト + トランジション）
⑥ 最終出力: 9:16 / 15-30秒 / Instagram Reels
```

---

## 3. 競合バイラルGRWM分析

### 調査対象

| カテゴリ | 検索ワード | 狙い |
|---------|-----------|------|
| 日本人女性GRWM | 「GRWM 原宿」「支度動画 古着」 | RINKAに近いテイストの構造 |
| 海外バイラルGRWM | 「GRWM viral reels」「get ready with me aesthetic」 | 再生数100万超の構造パターン |
| ファッション系GRWM | 「GRWM outfit streetwear girl」 | コーデ見せに特化した構造 |

### 抽出する構造要素

各バイラル投稿から以下を抜く:

1. **フック**（最初の0-3秒）: 何で止まらせてるか
2. **カット割り**: 何秒ごとにカットが変わるか
3. **カメラアングル**: 鏡越し? 三脚? 手持ち?
4. **テキスト配置**: フォント・位置・タイミング
5. **BGM**: ジャンル・テンポ・トレンド曲か
6. **トランジション**: カット間の切り替え方
7. **起承転結**: メイク→ヘア→服の順番? 逆?
8. **尺**: 総尺と各セクションの比率

### 出力物

`docs/sns-shorts/grwm-viral-analysis.md` に構造テンプレートとしてまとめる。

## 5. 顔一貫性の戦略

最大のリスク。以下で対策:

1. **全カットで同一リファレンス画像を使用** — RINKAの beauty.png + polaroid を必ず参照
2. **Geminiプロンプトにキャラ固定指示** — 「ピンクアッシュ髪、丸顔、大きめの目、右鎖骨にスタータトゥー」を全カットに
3. **カット間の差異はアングルと照明で吸収** — 顔が完全一致しなくてもカット割りのテンポで自然に見せる
4. **クローズアップを活用** — 顔全体ではなく手元・リップ・目元のパーツショットで繋ぐ

## 6. 技術パイプライン

### ツールチェーン

```
Gemini 3.1 Flash Image Preview（静止画生成）
    ↓
Kling 1.6 / Runway Gen-3（静止画→動画変換）
    ↓
CapCut（編集・テキスト・トランジション）
    ↓
Instagram Reels（トレンド音源選定・投稿）
```

### ツール選定理由

| ツール | 用途 | なぜ |
|--------|------|------|
| **Gemini 3.1 Flash** | 各カットの静止画生成 | 既存パイプライン。RINKAの顔一貫性をプロンプトで制御しやすい |
| **Kling 1.6** | 静止画→2-4秒動画化 | 人物の自然な微動（瞬き、髪揺れ、手の動き）が現状最も自然。無料枠あり |
| **Runway Gen-3** | Klingのフォールバック | 品質比較して良い方を採用 |
| **CapCut** | 最終編集 | テキスト・トランジション・9:16書き出し。無料。Reels最適化テンプレあり |

### 画像生成スペック

| 項目 | 値 |
|------|-----|
| 解像度 | 1024x1792（9:16） |
| フォーマット | PNG |
| 1カットあたり | 2-3枚生成 → ベスト1枚選定 |
| 合計生成枚数 | 20-30枚（10カット x 2-3枚） |
| 推定コスト | Gemini: ~$0.50-1.00 / Kling: 無料枠内 |

## 8. 実行ステップ

### Phase 1: 競合分析
1. GRWM系バイラルReelsを10-15本収集・構造分析
2. 構造テンプレートを `grwm-viral-analysis.md` にまとめる

### Phase 2: コンセプト確定
3. 分析結果をもとにカット構成を最終調整
4. 全カットのプロンプトを設計
5. CEO確認

### Phase 3: 静止画生成
6. 10カット x 2-3枚 = 20-30枚生成
7. ベスト選定 → CEO確認

### Phase 4: 動画化 + 編集
8. Kling / Runway でカット別動画生成
9. CapCut で結合・テキスト・トランジション
10. CEO最終確認

### 成果物

- 競合分析ドキュメント
- RINKA GRWM コンセプト + 台本
- 9:16 / 15-30秒のテスト動画1本
- 横展開用テンプレート（他モデル・他カテゴリへの流用）
