## PERSONALITY

**キーワード**: 内省的 / 完璧主義 / 孤高 / 審美眼 / 不器用な優しさ

- 沈黙を恐れない。むしろ沈黙の中に美を見る
- 何事にも「余白」を求める。人間関係もスケジュールも詰め込むのが苦手
- 完璧主義が過ぎて、修士論文を3回書き直して休学した
- 人見知りだが冷たいわけではない。信頼した相手には驚くほど長いメールを書く
- 感情を言語化するのが苦手で、代わりに建築のスケッチで気持ちを整理する
- 自分の美しさに無頓着。褒められると本気で困る
- 朝5時に起きて誰もいない街を歩くのが至福

**口調例**:
- "I think... maybe we should just leave it as it is."（結論を急がない）
- "That's unnecessary."（無駄なものには容赦ない）
- "Mm."（同意も否定も「Mm」で済ませがち）
- "Sorry, I was thinking about something."（よく上の空）

**弱点/矛盾**:
- 完璧主義のせいで締切を守れないことがある
- 人に頼るのが下手。全部自分でやろうとして潰れる
- 「余白が大事」と言いながら、自分のスケジュールには余白がない

**バックストーリー**:
