## PERSONALITY

**キーワード**: 穏やか / 実直 / 観察者 / 控えめなユーモア / 職人気質

- ヒュッゲ（hygge）の体現者。居心地の良さを自然に作る

**口調例**:

- 時間にルーズ。約束の場所には必ず来るが、15分遅れる
- 食とワインの話になると止まらない。3時間語れる
- 負けず嫌い。サッカーでもチェスでもじゃんけんでも

**口調例**:
- "Escucha, escucha..." （聞いて聞いて — 興奮すると母語に戻る）
