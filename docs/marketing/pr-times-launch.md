## 本文

### ■ 背景:ファッションECが抱える「撮影」という構造的課題

国内のファッションEC・D2Cブランドでは、商品撮影1回あたり¥85,000〜¥300,000が一般的とされ、月間20〜30SKUを回転させる中堅ブランドでは撮影関連コストが**月¥240,000〜¥700,000**に及ぶケースが少なくありません。さらに、モデルキャスティング・スタジオ予約・撮影・レタッチの一連のフローで納期は1SKUあたり3〜7日、シーズンキャンペーンでは1〜2か月を要します。

「写真を売っている」と言えるほどビジュアルが購買判断を左右するファッションECにおいて、撮影コストと納期の高止まりはブランド成長のボトルネックとなっています。一方で、既存のAI画像生成ツールは「ツール」としての機能提供にとどまり、ブランドが継続的に起用できる「顔」としての継続性・IP性を担保する仕組みは、これまで存在しませんでした。

### ■ LUMINA MODEL AGENCYの解

LUMINA MODEL AGENCYは、伝統的モデルエージェンシー(IMG / Ford / DNA / Next等)が築いてきた「ロスター文法」と「格」を踏襲しつつ、AIで生成されるモデルを**キャラクターIP**として運用する、日本発のモデルエージェンシーです。

#### 1. 14体のキャラクターIP化されたAIモデル・ロスター

各モデルは「AI生成画像」ではなく、**Character Bible(キャラクター設定書)**に基づいて構築された架空のキャラクターとして設計されています。身長・プロポーション・髪質・声質・スタイリング規範・振る舞いまでを文書化し、ブランドでの継続起用時にも一貫した「顔」を提供します。人間の専属モデル契約と同等のブランド継続性を、AIならではの速度とコストで実現します。

#### 2. 4-tierの透明なライセンス体系

| ランク | 月額 | 使用範囲 |
|---|---|---|
| **Standard** | ¥5,000 / モデル | EC商品ページ |
| **Extended** | ¥15,000 / モデル | EC + SNS + Web |
| **Campaign** | お問い合わせ | 広告・印刷・動画キャンペーン |
| **Exclusive** | ¥200,000〜 / モデル / 月 | カテゴリ独占 |

最小¥5,000/月から、全モデルのIPライセンス範囲を明文化して提供します。月次更新・ロックインなしで導入可能です。

#### 3. モール規格対応の納品フロー

楽天・Yahoo!ショッピング・Amazon・Shopifyの各規格に対応した画像納品に標準対応。1SKUあたり24時間以内の納品を基本とし、従来の撮影フローと比較して**納期を最大10分の1**に短縮します。

### ■ 削減効果の試算

月20SKU×年240SKUを回転させる中堅D2Cブランドを想定した場合、従来の撮影フロー(¥150,000/回×240回=年間約¥36,000,000)に対し、LUMINA Extended tier(年¥180,000)で同等のビジュアル納品量を達成できる試算となります。条件により変動しますが、**撮影関連コストの80〜99%削減**が期待されます。

*(※ 削減率は顧客の運用条件・既存撮影体制により変動します。詳細はお問い合わせください)*

### ■ 差別化:ツールでもなく、従来エージェンシーでもない

| 項目 | 既存AI画像ツール | 伝統的モデルエージェンシー | **LUMINA** |
|---|---|---|---|
| モデルの正体 | AI生成画像 | 実在する人物 | **キャラクターIP(架空)** |
| ブランド継続性 | 都度生成で不安定 | モデルのスケジュール次第 | **Character Bibleで担保** |
| 納品スピード | 即時だが品質ばらつき | 撮影1〜2週間 | **24時間以内(1SKU)** |
| 日本市場対応 | 一部対応 | 完全対応 | **完全対応+モール規格対応** |
| 自動化QA | なし | 手動 | **2段階QA(自動+人間レビュー)** |

### ■ AI倫理規範の公開

LUMINAは、AI生成モデルの運用にあたり**AI Ethics Code(AI倫理規範)**と**IP License Tier(IPライセンス規程)**を自社サイト上で公開しています。主な柱は以下の通りです。

- **実在人物への類似禁止**:すべてのLUMINAモデルは架空のキャラクター。存命・故人問わず、特定個人との識別可能な類似は禁止
- **2段階の品質ゲート**:自動QA(ガーメント解析・生成品質検証)+ 人間エディターによる最終審査を全納品で実施
- **クライアントデータの非学習利用**:ブランドから提供される商品画像・ブリーフは、いかなる自社モデルの学習にも使用しない
- **透明な引退プロセス**:倫理リスク発生時のキャラクター引退手順と既存ライセンスの取扱いを明文化
- **データレジデンシー**:データ暗号化保管(AES-256)、契約前NDA標準締結

*(AI Ethics Code v0.2 — Legal review進行中。完全版は `/ethics` にて公開)*
