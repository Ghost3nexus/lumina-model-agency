## アルゴリズム重み付け（2026年版）

| アクション | 重み（Like=1x基準） | 戦略的意味 |
|-----------|---------------------|-----------|
| Reply + Author Response | **150x** | 最重要。返信を誘発し、必ず返信する |
| Reply（単独） | **27x** | 返信されやすい問いかけ・論争を仕込む |

## フックの3要素（全投稿の冒頭に必ず1つ以上）

> Source: おさるハイブリッドマーケティング + PASONAの法則

| 要素 | 効果 | 例 |
|------|------|-----|
| **短期的快楽** | 「これを知れば得する」 | 「月100万円削減できる方法」 |
| **逆説/衝撃の事実** | 「常識と違う」で注意を引く | 「毎日投稿すればフォロワー増える、は嘘」 |
| **ネガティブ** | 損失回避で行動させる | 「知らないと月50万円損してる」 |

**ルール**: タイトル・冒頭15-30字に必ずこの3要素のどれかを入れる。
これはX投稿だけでなく、note記事タイトル・ブログタイトル・動画サムネにも適用。

## 投稿テンプレート（2026年アルゴリズム最適化）

### Template 1: 会話誘発型（Reply+Response = 150x狙い）

```
【逆説 or 論争的主張】（15-30字）

[根拠・具体例を2-3行]

これ、どう思う？
[選択肢A] or [選択肢B]？

返信で教えて👇
```

**狙い**: Reply を最大化。投稿者が全リプに返信することで150xブースト
**ルール**: 必ず30分以内に全リプに返信する

### Template 2: 保存型スレッド（Bookmark + Thread = 30x狙い）

```
🧵 [テーマ]を完全解説する（保存推奨）

1/ [衝撃的な導入]
[具体数字・データ]

2/ [ポイント①]
[具体例]

3/ [ポイント②]
[具体例]

4/ [ポイント③]
[具体例]

5/ まとめ
→ [一言結論]
→ 保存して見返してください

（画像 or 図解を各ツイートに添付）
```

**狙い**: BM率最大化 + スレッドの3x + 画像の4.4x
**文字数**: 各ツイート150-200字

### Template 3: 動画フック型（Video = 10x狙い）

```
[衝撃的な一文]（15字以内）

[動画: 30-60秒]
- 0-3秒: フック（数字 or ビジュアルインパクト）
- 3-15秒: 問題提起
- 15-45秒: 解決 or 裏側を見せる
- 45-60秒: CTA（フォロー or 保存）

↓の動画で全部わかる
```

**狙い**: 動画10x + Reply誘発
**動画仕様**: 縦型(9:16) or 横型(16:9)、字幕必須、冒頭3秒が命

### Template 4: 引用誘発型（Quote = 25x狙い）

```
【断言】[強い主張]。

[根拠を1-2行]

異論ある人は引用で反論してほしい。
間違ってたら訂正する。
```

**狙い**: Quote Tweet 25x。あえて論争を呼ぶ主張で引用を誘発
**注意**: 炎上ではなく「建設的な議論」に持っていく

### Template 5: データ図解型（Image + Bookmark = 14.4x狙い）

```
[タイトル]を図にした。

[図解画像: 1200x675 or 1080x1080]
- インフォグラフィック
- 比較表
- フロー図
- チートシート

保存して使ってください。
```

**狙い**: 画像4.4x + BM10x。「保存価値」を最大化

### Template 6: ロングフォーム考察（Long-form = 2x狙い）

```
[テーマ]について、本音で書く。

[500-1000字の考察]
- パラグラフ1: 問題提起
- パラグラフ2: 自分の経験
- パラグラフ3: データ or 具体例
- パラグラフ4: 結論 + 問いかけ

あなたはどう考えますか？
```

**狙い**: Long-form 2x + Reply誘発。深い共感でフォロー転換

## リンク戦略（外部リンクペナルティ回避）

❌ NG: 本文にURLを貼る（50-90%ペナルティ）

✅ OK:
1. **リプライにリンク**: 本文は完結させ、リプで「詳しくはこちら→[URL]」
2. **プロフィールリンク**: 「プロフのリンクから」で誘導
3. **引用RTにリンク**: 別の投稿で引用しつつリンク追加
4. **画像にURL記載**: スクリーンショットやカード画像内にURL

### note/ブログへの誘導フロー

```
[X投稿: フック + 核心]
  ↓ リプライ
[「続きはnoteで書きました → [URL]」]
  ↓ note記事内
[「もっと詳しくはブログで → [blog URL]」]
  ↓ ブログ内
[CTA → LP / サービスページ]
```

## 禁止事項

1. ❌ 本文に外部リンクを直接貼る
2. ❌ 「〇〇かもしれない」「〇〇と思う」 → 断言する
3. ❌ テキストのみ投稿（画像・動画なしは原則禁止）
4. ❌ リプを放置する（30分以内に返信必須）
5. ❌ 1日1投稿で終わる（最低2投稿 + リプ活）
6. ❌ 全投稿を同じテンプレで作る（6テンプレをローテーション）
7. ❌ ハッシュタグの乱用（0-2個まで。X では不要）
