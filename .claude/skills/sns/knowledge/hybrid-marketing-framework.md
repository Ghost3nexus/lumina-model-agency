2. **スクリーニング**: 冷やかし・アンチ層を自動排除

## コンテンツ戦略の鍵

### フロントで圧倒的価値提供

> 「情報は無料で全て出してOK」

- 有料級の情報を無料で出す → 権威性UP → 高単価商品が売れる
- 出し惜しみしない。むしろ「こんなに無料で出すの？」が信頼を生む
- **無料と有料の差は「情報」ではなく「体験」と「サポート」**

### サムネイル/フックの3要素

全コンテンツの冒頭（X投稿、ブログタイトル、動画サムネ）に必ず1つ以上:

1. **短期的快楽**: 「これを知れば〇〇できる」
2. **逆説/衝撃の事実**: 「〇〇は嘘だった」「実は〇〇」
3. **ネガティブ**: 「〇〇しないと失敗する」「知らないと損する」

### 非言語の重要性

> 「非言語を操った人が最高の発信者になる」

- **何を言うか** より **誰が言うか**
- 権威性・実績の提示は必須（冒頭で）
- ビジュアル品質（ダーク×ミニマル×テック）で差別化

### 高単価商品戦略

- **ミドル商品の安売りは避ける**
- ファン化 → 相場破壊 → 適正（高）価格で販売
- LTV理論よりも「1回の高単価」が確実

## X投稿への統合ルール

1. **全投稿がファネルの入口** — 必ず次のステップへ誘導
2. **フックの3要素**を冒頭に（短期的快楽/逆説/ネガティブ）
3. **権威性を示す** — 実績数字・具体的成果を必ず入れる
4. **出し惜しみしない** — 有料級の情報を無料で出す
5. **スクリーニング意識** — 万人受けより「刺さる人に深く刺す」

## 動画コンテンツ制作指針

### 構成テンプレート（YouTube/リール共通）
1. **フック**（0-15秒）: 衝撃数字 or 逆説 + 権威性
2. **問題提起**（15-30秒）: 視聴者の痛みに共感
3. **本編**（30秒-）: 具体的ノウハウ（2倍濃く、分かりやすく）
4. **CTA**（末尾）: 次のステップへ誘導（LINE/note/ブログ）

### 動画テクニック（重要なもの抜粋）
- **冒頭15-30秒で50%離脱** → フックが命
- **権威性は毎回提示** — 「〇〇を達成した方法」
- **2倍濃く・分かりやすく** — 他と同じ情報でも再構成で価値UP
- **伸びたコンテンツの同系統を作る** — ネタ切れ防止
- **切り口を変えてリメイク** — 過去コンテンツの再活用
