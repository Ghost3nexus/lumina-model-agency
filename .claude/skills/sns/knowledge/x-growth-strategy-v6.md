| アクション | 重み（Like=1x基準） | 意味 |

| ブックマーク | **10x** | 「保存したい」＝深い関心シグナル |

## 柱1: リプライ戦略（最重要・新規追加）

### ターゲットアカウント（日本語AI/テック圏）

毎日以下のアカウントの新投稿をチェックし、価値あるリプライを付ける:

#### Tier 1（万フォロワー・AI業界リーダー）

- @AIBunCho（AI文章生成）

- @ochyai（落合陽一 / テクノロジー）
- @0xMidnight（AIエージェント）

#### Tier 2（1000-1万・成長中のAI実践者）
- AIスタートアップCEO
- AIツールレビュアー
- ソロプレナー・1人法人アカウント

#### Tier 3（同規模・相互成長）
- AI活用している個人事業主
- プログラミング×AI実践者

### リプライの型（5テンプレート）

| 型 | 名前 | 使い方 | 例 |
|---|------|--------|---|
| R1 | **データ追加** | 相手の主張に具体的な数字を追加 | 「うちで実測したら〇〇だった。体感と一致する」 |
| R2 | **実体験補強** | 自社のAI運用体験で裏付ける | 「22体運用してて全く同じ問題に当たった。うちは〇〇で解決した」 |
| R3 | **建設的反論** | 敬意を持って別視点を提示 | 「面白い視点。ただ〇〇の観点だと逆もあり得る」 |
| R4 | **質問深掘り** | 相手の専門知識を引き出す質問 | 「これって〇〇の場合はどうなります？」 |
| R5 | **事例共有** | ミニケーススタディを1-2文で | 「似た事例: うちのCFO Agentが〇〇した結果〇〇」 |

### リプライの鉄則

1. **投稿後2時間以内にリプライ** — アルゴリズムの鮮度ウィンドウ内
2. **「いいですね！」禁止** — 必ず独自の情報・視点・データを追加
3. **自社宣伝禁止** — 価値提供のみ。相手が興味を持てばプロフを見に来る
4. **1日最低20リプライ** — うち10本はTier 1アカウント宛
5. **返信が来たら即レス（30分以内）** — 150xブーストを逃さない

### リプライ探索フロー

```
毎朝:
1. Tier 1アカウントの最新投稿を確認（直近12時間）
2. AI/エージェント関連のトレンドワード検索
3. 自社投稿への返信を全て確認→即レス

毎昼:
4. 午前中のバズ投稿をチェック→リプライ追加
5. 自社投稿のエンゲージメント初速確認

毎夕:
6. 夕方のニュース投稿にリプライ
7. 1日のリプライ数を記録
```

## 柱2: 引用RT戦略（新規追加）

### 自己引用RT（セルフQT）
- 自分のオリジナル投稿を**4-6時間後**に引用RTする
- 追加コメント: より過激な意見、後日談、データ追加
- → 同じ投稿のリーチを**2倍**にする

### 他者引用RT
- バズっているAI関連投稿に**自社視点を乗せて**引用RT
- 「うちではこうしている」「この数字の裏側はこうだ」
- → 相手のオーディエンスに自分の視点が届く

### 引用RTの型

| 型 | 名前 | 例 |
|---|------|---|
| Q1 | **実体験追加** | 「[QT] うちの22体AIでも全く同じ。特にCFO Agentが〇〇」 |
| Q2 | **反論・別角度** | 「[QT] これ半分正しいけど、実運用だと〇〇が抜けてる」 |
| Q3 | **データ補強** | 「[QT] ちなみにうちの実測値: 〇〇。公式データと〇〇%乖離」 |
| Q4 | **セルフQT更新** | 「[自分の投稿をQT] あれから3日、結果が出た→〇〇」 |

## 柱3: オリジナル投稿（v5.0を継続・量を調整）

### 変更点

| 項目 | v5.0 | v6.0 |
|------|------|------|
| 本数/日 | 10本 | **3-5本** |
| 配分 | Type A×5 + B×5 | Type A×2 + B×2 + 引用RT×1 |
| 時間帯 | 10枠固定 | **3枠（朝・昼・夕）+ 臨時** |
| 画像 | 全投稿必須 | 全投稿必須（維持） |
| スレッド | 2本まで | 週1本は**5-7ツイートの深掘りスレッド** |

### 3枠タイムスロット

| 枠 | 時刻 | 最適タイプ | 狙い |
|----|------|----------|------|
| 朝 | 07:00-08:00 | Type A | 通勤トレンド速報 |
| 昼 | 12:00-12:30 | Type B or A | 昼休みピーク |
| 夕 | 18:00-19:00 | Type A or B | 帰宅ピーク・煽り |

### 週1スレッド（木曜推奨）

- 5-7ツイート構成の深掘りスレッド
- テーマ: AI運営の裏側、実装ハウツー、データ公開
- スレッドは**3xエンゲージメント**（アルゴリズム優遇）
- → ブックマーク率最大化
