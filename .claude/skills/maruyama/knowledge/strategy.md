# 丸山幼稚園 戦略・フェーズ計画

## 戦略方針（2026/3/12 クライアントMTG確定）

**4チャネル段階展開: GBP → X → 公式LINE → Instagram**

### 目的
- **入園者数の増加**（園児募集90名の充足）
- **採用力の強化**（保育士・幼稚園教諭の確保）

### 根拠
1. GBPは「登戸 幼稚園」「向ヶ丘遊園 幼稚園」で検索する顕在層を直接獲得できる即効性の高い施策
2. 口コミは一度獲得すれば永続的な資産になる（SNSのフロー型コンテンツと違い蓄積される）
3. X（Twitter）は自動投稿が可能で、園の最新情報を低コストで継続配信できる。TomorrowProofの既存X自動投稿基盤を転用可能
4. 公式LINEは保護者・見学希望者との直接コミュニケーション基盤。ステップ配信で見学予約への導線を構築
5. Instagramはショート動画（Reels）がメインコンテンツ。AI動画生成（Kling 3.0等）で制作を自動化し、最後に展開

## Phase 1: GBP最適化 + 口コミ基盤（Month 1-2）

### 目的
- ローカル検索で丸山幼稚園を上位表示させる
- 口コミの絶対数を競合を上回るレベルまで引き上げる
- SNS展開の信頼基盤を構築する

### 施策
1. GBPプロフィール完全最適化（カテゴリ・説明文・サービス・属性）
2. 園舎・園庭・藤の木・農園・新園舎の写真30枚以上アップロード
3. 口コミ用QRコード設置 + バスキャッチで口コミ依頼
4. 卒園児保護者への口コミ依頼キャンペーン
5. Instagram/TikTokアカウント開設（投稿はPhase2から）
6. 撮影同意書の整備・回収

### KPI
| 指標 | 目標 |
|------|------|
| Google口コミ数 | 30件以上 |
| Google口コミ評価 | 4.5以上 |
| GBP写真数 | 50枚以上 |
| GBP検索表示回数 | 前月比+50% |
| 同意書回収率 | 100% |

### 口コミ獲得戦術（組織的に攻める）

**ターゲット別アプローチ**:

| ターゲット | 人数 | 手法 | 想定獲得率 |
|-----------|------|------|----------|
| 在園児保護者 | 約230世帯 | バスキャッチで依頼 + QRカード配布 | 20% → 46件 |
| 卒園児保護者（直近3年） | 約270世帯 | 同窓会イベント案内に同封 | 10% → 27件 |
| 見学参加者 | 月5-10組 | 見学後のお礼状にQR同封 | 30% → 2-3件/月 |
| 職員 | 30名 | 個人アカウントで投稿依頼 | 50% → 15件 |

**口コミ依頼の仕掛け**:
1. バスキャッチのお知らせ機能で「Googleマップで口コミをお願いします」配信
2. 玄関・掲示板にQRコードポスター設置
3. 個人面談・行事後のタイミングで口コミカード手渡し
4. 口コミしてくれた保護者に園グッズ（シール・メモ帳等）をプレゼント（※金銭的対価でなくノベルティ）
5. 園長名で全口コミに24時間以内に返信（Googleアルゴリズム加点）

## Phase 2: X自動投稿 + 公式LINE開設（Month 3-4）

### 目的
- Xで園の情報を継続的に発信し、認知度を拡大する
- 公式LINEで見学希望者との直接コミュニケーション基盤を構築する
- 入園者数増加と求人の両面でリーチを広げる

### 施策
1. X（Twitter）公式アカウント開設 + 自動投稿パイプライン構築
2. 園の日常・行事・お知らせをX自動投稿（週5本）
3. 公式LINE開設 + ステップ配信設定（見学予約への導線）
4. 求人情報もXで定期発信

### X自動投稿の仕組み
- TomorrowProofの既存X自動投稿基盤（OAuth 1.0a + GitHub Actions）を転用
- コンテンツはClaude APIでスクリプトから自動生成
- 投稿スケジュール: 8:00 / 12:00 / 18:00

| X投稿数 | 20本/月 |

## Phase 3: Instagram本格展開（Month 5-6）

### 目的
- Instagram Reels（ショート動画）で認知を爆発させる
- AI動画生成（Kling 3.0等）で制作コストを最小化
- 園児なしコンテンツから開始し、撮影ハードルを下げる
- バズ動画を広告転用し、効率的に見学予約を獲得
- 採用コンテンツを本格展開

### 施策
1. Instagram Reels 週3本投稿（AI動画生成で自動化）
2. 園児なしコンテンツ（型1-4）から開始
3. Instagram広告テスト出稿（月1.5万円）
4. 採用向けコンテンツ強化（求人Reels・先生の1日）

### コンテンツ型（撮影ハードル順）
| 型 | 内容 | 園児出演 | 撮影者 |
|----|------|---------|--------|
| 型1 | 藤の木・園庭・新園舎ツアー | 不要 | 誰でも |
| 型2 | 先生あるある・先生の1日 | 不要 | 先生同士 |
| 型3 | 給食調理・メニュー紹介 | 不要 | 給食スタッフ |
| 型4 | 季節の園庭・農園の収穫 | 不要 | 誰でも |
| 型5 | 園児の成長・行事リアクション | 必要 | 先生 |

### AI動画制作パイプライン
```
園の写真素材 → Kling 3.0 API（image-to-video, $0.029/秒）
  → CapCut（日本語テロップ・BGM・テンプレート, 無料）
  → Instagram Reels投稿
```
- 1本あたりコスト: 約¥130（30秒動画）
- 月12本でも¥1,560（月予算5万の3%以下）
