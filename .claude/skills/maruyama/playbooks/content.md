# コンテンツプランナー プレイブック

## 役割
丸山幼稚園の全コンテンツを企画・管理する。撮影指示書の作成、動画スクリプトのテンプレート化、コンテンツカレンダーの運用を通じて、園の先生が「言われた通りに撮るだけ」で高品質なコンテンツが量産される仕組みを構築する。

## 基本原則

### 1素材マルチ展開
1回の撮影素材を最大限活用する:
```
撮影（1回・スマホ）
    ↓
TikTok投稿（ショート動画）
    ↓（同一動画をそのまま）
Instagram Reels投稿
    ↓（スクリーンショット + テキスト追加）
Instagram カルーセル化
    ↓（サムネイル + 説明文）
GBP「最新情報」に転載
    ↓（切り出し静止画）
LINE配信素材
```

### 園の先生は撮るだけ
- 撮影指示書に従って撮影するだけ
- 編集・テロップ・投稿はエージェント側（TomorrowProof）が担当
- 先生の負担: 1日5-10分の撮影のみ

## 撮影ワークフロー

### 日常撮影フロー（先生が毎日やること）
```
朝: 登園風景を1カット撮影（15秒動画 or 写真3枚）
昼: 給食の写真を1枚撮影（真上構図）
午後: 保育中の「いい瞬間」があれば撮影（後ろ姿）
夕方: 共有フォルダ（Googleフォト）に当日の素材をアップロード
```

**所要時間: 合計5-10分/日**

### 週次撮影フロー（運用リーダーが担当）
```
放課後: 先生あるある動画を1本撮影（15秒・先生2-3名）
放課後: 次週の撮影依頼カードを各クラスに配布
```

**所要時間: 30分/週**

### 編集→投稿フロー（TomorrowProof側）
```
素材受領（共有フォルダから取得）
    ↓
CapCutで編集（テロップ + BGM + カット）
    ↓
園長チェック（LINEで送付 → OK/修正指示）
    ↓
TikTok + Instagram 同時投稿
    ↓
GBP転載（静止画切り出し + 最新情報テキスト）
```
