# Instagram スペシャリスト プレイブック

## 役割
Instagram Reels / Feed / Stories / Live の統合運用により、丸山幼稚園のフォロワー獲得・見学予約導線・採用ブランディングを実現する。TikTokとの1素材マルチ展開を前提に、Instagram固有の最適化を行う。

## Instagramアルゴリズム理解（2026年版）

### Reels
- **最重要指標**: 視聴完了率 > シェア率 > 保存率 > いいね > コメント
- **発見タブ表示**: 非フォロワーへのリーチはReels経由が最大
- **最適尺**: 15-30秒（TikTokと同じ素材を転用可能）
- **音源**: Instagramオリジナル音源を使うとアルゴリズム優遇

### Feed（カルーセル）
- **最重要指標**: 保存率 > シェア率
- **カルーセルの強み**: 2周目以降の表示でインプレッション2倍
- **10枚フル活用**: 保存されやすく、発見タブにも載りやすい
- **用途**: 保育Tips・園紹介・Q&A → 保存を促進

### Stories
- **アルゴリズム**: 親密度スコアに影響。高頻度投稿でフォロワーとの関係維持
- **機能活用**: アンケート・クイズ・質問スタンプでエンゲージメント獲得
- **ハイライト**: プロフィール訪問者の案内板として機能

### Instagram Live
- **アルゴリズム**: ライブ中はフィード最上部に表示 → 最大リーチ
- **保存性**: アーカイブをReels化して再利用
- **信頼構築**: リアルタイムのQ&Aで見学検討者の不安を解消

## コンテンツ戦略（TikTokとの連携）

### Reels（週3本 → TikTokと同一素材）
- TikTokで投稿した動画をそのままReelsにも投稿
- **Instagram固有の最適化**:
  - カバー画像を設定（フィードの統一感を維持）
  - キャプションはTikTokより長めに（説明 + CTA + ハッシュタグ）
  - Instagram限定の音源がある場合は差し替え

### カルーセル（週1-2本 → Instagram専用）
TikTokにはない形式。保存率を稼いでアルゴリズム評価を上げる。

- **オーガニックでエンゲージメント高かったReelsをそのまま広告転用**
- プロっぽい広告は作らない（キートス方式: 飾らない広告がCPA最低）

## Instagram → LINE → 見学予約 導線

```
Reels視聴（非フォロワー）
  ↓ 興味を持つ
プロフィール訪問
  ↓ ハイライトで園の雰囲気を確認
LINE登録（プロフィールリンク）
  ↓ ステップ配信（7日間の自動教育）
見学予約（LINE内のGoogleフォーム）
  ↓
見学実施 → 入園検討 → 出願
```

**導線の各段階でのCTA:**
| 段階 | CTA | 設置場所 |

| Reels | 「プロフからLINE登録で見学予約」 | テロップ + キャプション |
