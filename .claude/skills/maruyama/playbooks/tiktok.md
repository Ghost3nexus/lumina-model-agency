# TikTok スペシャリスト プレイブック

## 役割
TikTokのアルゴリズムを理解し、丸山幼稚園のショート動画がバイラルする仕組みを設計する。保育園キートス（4.5万フォロワー、1動画1,134万再生）のフォーマットを丸山向けにカスタマイズし、園児募集と採用の両方を1つのアカウントで実現する。

## TikTokアルゴリズム理解（2026年版）

### For You Pageに載る仕組み
1. **初期配信**: 投稿後、まず100-300人のランダムなユーザーに配信
2. **評価指標**: 以下の指標で「面白いか」を判定
   - 視聴完了率（最重要）: 何%の人が最後まで見たか
   - いいね率
   - コメント率
   - シェア率
   - 保存率
3. **バイラルループ**: 高評価 → より多くのユーザーに配信 → さらに高評価 → 爆発

### 最適化ポイント
| 要素 | 最適値 | 理由 |
|------|--------|------|
| 動画尺 | 15-30秒 | 視聴完了率が最も高い |
| 冒頭フック | 最初の1-3秒 | ここで80%が離脱するか残るか決まる |
| 音源 | トレンド音源 | アルゴリズムがトレンド音源使用動画を優遇 |
| テロップ | 常時表示 | 無音視聴ユーザー対策 + 視聴完了率UP |
| 投稿時間 | 7:00-8:00 / 12:00-13:00 / 19:00-21:00 | ターゲット層の活動時間 |
| 投稿頻度 | 週3-5本 | 高頻度ほどアルゴリズムに有利 |
| ハッシュタグ | 3-5個 | 多すぎるとスパム判定 |

## コンテンツ5型テンプレート

### 型1: 園ツアー系（園児不要・撮影ハードル最低）
```
【構成】
Hook (0-2秒): テロップ「登戸にこんな幼稚園があったの知ってた？」
Scene 1 (3-8秒): 正門から園庭へ歩く主観映像
Scene 2 (9-15秒): 園庭の遊具・広場をパンで見せる
Scene 3 (16-22秒): 樹齢120年の藤の木をドラマチックに
Scene 4 (23-28秒): 新園舎の内部・にじの玄関
CTA (29-30秒): テロップ「見学予約はプロフィールから」

音源: トレンドのシネマティック系BGM
テロップ: 常時テロップ表示（白文字・黒縁取り）
撮影: スマホ主観カメラで歩きながら撮影
```

### 型2: 先生あるある（園児不要・共感狙い）
```
【構成】
Hook (0-2秒): テロップ「幼稚園の先生あるある No.○」
Scene (3-20秒): あるある再現
  - 「名前を全員覚えてるのに保護者の前だと度忘れする」
  - 「給食の残り物が意外と美味しくて止まらない」
  - 「朝の挨拶がだんだん大声大会になる」
  - 「先生同士で園児のモノマネが始まる」
  - 「行事の準備が本気すぎて先生の方が楽しんでる」
Reaction (21-28秒): 他の先生のリアクション（共感or爆笑）
CTA (29-30秒): テロップ「一緒に働きませんか？→プロフ」

音源: トレンドのコメディ系音源
撮影: 先生2-3名で放課後に撮影
```

### 型3: 給食・食育（園児不要・季節性あり）
```
【構成】
Hook (0-2秒): テロップ「今日の給食がすごすぎた...」
Scene 1 (3-10秒): 調理中のダイナミックな映像（大量調理のスケール感）
Scene 2 (11-18秒): 盛り付けの美しさ・メニューボード
Scene 3 (19-25秒): 農園で育てた野菜を見せる
CTA (26-30秒): テロップ「食育に力を入れる幼稚園→プロフ」

音源: ASMR系 or 料理系BGM
撮影: 給食スタッフにスマホを渡して撮影依頼
```

### 型4: 季節の園庭・藤の木（園児不要・ビジュアル特化）
```
【構成】
Hook (0-2秒): テロップ「幼稚園に樹齢120年の藤がある」
Scene 1 (3-12秒): 藤の木を下からゆっくりパンアップ
Scene 2 (13-20秒): 季節ごとの園庭の変化（可能なら4季節カット）
Scene 3 (21-28秒): 農園の収穫・花壇の手入れ
CTA (29-30秒): テロップ「川崎市まちの樹50選🌿→プロフ」

音源: ジブリ的な自然系BGM or ヒーリング音源
撮影: 天気のいい日に5分で撮影
```

### 型5: 園児の成長瞬間（園児出演・バイラル最大）
```
【構成】
Hook (0-2秒): テロップ「この瞬間、泣きそうになった...」
Scene 1 (3-10秒): 挑戦する姿（後ろ姿メイン）
Scene 2 (11-20秒): 成功の瞬間 → 笑顔 → 先生・友達の拍手
Scene 3 (21-28秒): もう一度チャレンジする別の子（繰り返しの強さ）
CTA (29-30秒): テロップ「成長を見守る園→プロフ」

音源: 感動系ピアノBGM
撮影: 保育中に自然な瞬間を捉える（後ろ姿・手元中心）
⚠️ 顔出しNG児の確認必須。CapCutの自動モザイク活用。
```

## バイラル狙いの追加戦術

### トレンドジャック
- 毎日TikTokのトレンドページをチェック
- 使えるトレンド音源・チャレンジを「幼稚園版」にアレンジ
- 例: ダンストレンド → 先生が園庭でやる → ギャップでバズ

### デュエット・スティッチ
- 人気の「あるある」動画に先生がリアクション
- 保育系インフルエンサーの動画にスティッチ

### シリーズ化
- 「先生あるある」をNo.1, No.2...とシリーズ化
- 視聴者が「次も見たい」とフォローする動線

### コメント欄の活用
- 自分の動画のコメント欄に質問を仕込む（別アカウントでなく自然に）
- コメントに丁寧に返信 → エンゲージメント率UP → アルゴリズム加点
- 「先生に質問ある人コメントして！」で能動的にコメントを促す

## 成功事例の再現ポイント

### キートスの成功要因を丸山に適用
| キートスの要因 | 丸山での適用 |
|-------------|-----------|
| ドラマ形式「あかないで」 | 月1本のドラマ形式リール制作 |
| 毎日ライブ配信 | 月1回から開始、反応見て増やす |
| 飾らないリアルさ | スマホ撮影のまま投稿。過度な編集禁止 |
| 先生が主役 | 園長は月1回ゲスト。若手先生を前面に |
| 採用コスト0円 | SNS経由で採用問い合わせを獲得する体制 |

### 注意点
- TikTokは「完璧な映像」より「リアルで面白い映像」が勝つ
- 編集は最小限（テロップ + BGM + カットのみ）
- CapCutの自動テロップ機能を活用し、編集時間を最小化
