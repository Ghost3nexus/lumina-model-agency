# 会長向け月次報告書テンプレート（`!report` コマンド）

**対象読者**: 株主・会長（75歳）。ITやAIの専門用語は一切使わない。
**鉄則**: 数字を主体にする。文章は最小限。表と箇条書きで伝える。

## 絶対ルール
- **専門用語禁止**: API→「AIツール利用料」、ROAS→「広告の費用対効果」、CVR→「成約率」、サーバー→「システム運営費」等に言い換える
- **金額は万円単位**: 細かい端数は四捨五入。1万円未満は「約○千円」
- **前月比を必ず記載**: 数字は単独ではなく必ず前月との比較を添える
- **1ページ（2000文字以内）**: 長い報告書は読まれない。簡潔に
- **日本語のみ**: 英語の社名・用語も可能な限り日本語で説明を添える

## データ収集元（各エージェントから取得）
| データ | 取得元 | コマンド |
|--------|--------|---------|
| 売上・経費・利益・残高 | CFO Agent | `!cfo PL` / `!cfo キャッシュ` |
| 広告費・問い合わせ数 | Google Ads Agent | `!ads` |
| 営業進捗・新規顧客 | Sales Agent | `!sales` |
| 開発進捗 | Dev Agent | `!dev` |
| 事業全体の方針 | CEO（CLAUDE.md） | — |
