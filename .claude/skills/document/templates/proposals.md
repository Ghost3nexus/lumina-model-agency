# Document Agent テンプレート集

## 提案書テンプレート（PREP法）

```
## [提案タイトル]

**作成日**: YYYY年MM月DD日
**作成者**: TomorrowProof

---

### エグゼクティブサマリー（Point: 結論）
（3行以内で提案の核心を述べる）

### 1. 背景・課題（Reason: 理由）
（なぜこの提案が必要か。データで裏付ける）

### 2. 提案内容（Example: 具体）
（具体的に何を提案するか）

### 3. 期待効果
（定量的な効果見込み）

### 4. 実施スケジュール
| フェーズ | 期間 | 内容 |
|---------|------|------|

### 5. 費用
| 項目 | 金額 |
|------|------|

### 6. 結論（Point: 結論再掲）
（改めて提案の核心と次のステップ）

### 会社概要
TomorrowProof / 代表: KOZUKI TAKAHIRO
```

## ピッチデック構成（10枚）

```
1. タイトル（会社名・タグライン）
2. 課題（市場のペインポイント）
3. 解決策（プロダクト概要）
4. 市場規模（TAM/SAM/SOM）
5. プロダクト（スクリーンショット・デモ）
6. ビジネスモデル（収益構造）
7. トラクション（実績・数字）
8. 競合優位性（ポジショニングマップ）
9. チーム（代表プロフィール）
10. Ask（依頼事項・次のステップ）
```

!doc 議事録 [顧客名]          → 壁打ち議事録作成
!report 会長                  → 会長向け月次報告書を作成
