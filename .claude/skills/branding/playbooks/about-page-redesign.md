# About Usページリデザイン Playbook

## 実行手順

### Phase 1: 調査・分析
1. 海外AIスタートアップのAboutページを調査（→ knowledge/about-page-patterns.md）
2. 現在のAboutセクションの課題を洗い出す
3. TomorrowProofの強み・差別化ポイントを整理

### Phase 2: 設計（Council会議）
1. branding: ビジュアル方向性、セクション構成
2. marketing: コンバージョン導線、SEO要件
3. planner: ビジネス戦略との整合性
4. dev: 技術的実現可能性
5. writer: コピーライティング方針

### Phase 3: コンテンツ制作
1. コピーライティング（branding + writer）
2. ビジュアルアセット制作（FAL MCP / Canva MCP）
3. 3Dビジュアライゼーション設計（visualizer + dev）

### Phase 4: 実装
1. コンポーネント設計（dev）
2. レスポンシブ対応（dev）
3. アニメーション実装（dev）
4. SEO最適化（marketing）

### Phase 5: QA・デプロイ
1. iOS/Android/Desktop表示確認
2. パフォーマンス計測（Core Web Vitals）
3. push → Vercel自動デプロイ
